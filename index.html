A high performance, open source, general RPC that puts mobile and HTTP/2 first.

Source code on GitHub: grpc, grpc-java, grpc-go.

Powerful IDL

Define your service using Protocol Buffers, a powerful binary serialization toolset and language with support for canonical mapping to REST. 

message HelloRequest {
  string greeting = 1;
}

message HelloResponse {
  string reply = 1;
}

service HelloService {
  rpc SayHello(HelloRequest) returns (HelloResponse) {}
}

Generate Clients and Servers

Automatically generate idiomatic client and server stubs for your service in a variety of languages.

gRPC currently supports C, Java and Go with a host of additional languages layered on top of the C core:
C++, Node.js, Python, Ruby, Objective-C, PHP and C#

HTTP/2

Building on HTTP/2 standards brings many capabilities such as bidirectional streaming, flow control, header compression, multiplexing requests over a single TCP connection and more.

These features save battery life and data usage on mobile while speeding up services and web applications running in the cloud.
