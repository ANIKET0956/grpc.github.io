Where are the docs?

Since gRPC supports so many languages and the project is so new, the docs you might be interested in are a little tricky to find. Read on...

Shared Overview & Examples (grpc-common)

Contains shared documentation, examples and example services (Proto files) which apply to all languages. There is an overview document and language specific hello world tutorials in each language subfolder.

grpc

Contains the C core source and C based language support: {% include lang-list.html %}

grpc-java

Contains the Java implementation and Java specific examples.

grpc-go

Contains the Go implementation and Go specific examples.
