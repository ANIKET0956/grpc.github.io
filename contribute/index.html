How to Contribute

To contribute to gRPC, please fork the GitHub gRPC repository which contains the language or runtime you are interested in contributing to and start submiting pull requests.

github.com/grpc/grpc

grpc-go

github.com/grpc/grpc-go

For the Go implementation.

IRC Channel

Join us on the #grpc IRC channel on Freenode to talk to other developers working on the project.

Mailing list

Join us on Google Groups. Email: grpc-io@googlegroups.com
